# SmartHealth-AI
Health Insurance Cost Prediction App  This web app, which is built with the Streamlit web development framework, uses a machine learning regression algorithm for the purpose of health insurance cost predictions, taking into consideration the personal and lifestyle characteristics of the individual.
The model consists of a Random Forest Regressor, which has the ability to handle non-linear relationships. The user can enter the data using the interactive interface. The estimated insurance cost can be obtained immediately.

**Key Features :**

Interactive Streamlit UI for user-friendly input.

**Random Forest Regression Model for Accurate Predictions**

**Real-time Cost Estimation**

Simple and clean interface, easy to use by non-technical users

Machine Learning Workflow

Data Preprocessing (Handling Numerical & Categorical Data)

**Model training using Scikit-learn**

Model evaluation to validate the level of accuracy in its predictions

Deployment using Streamlit

**Input Features :**

The model bases its predictions on the following commonly used insurance factors:

Age

Gender

BMI

Number of dependents

Smoking status
